# 🥷 Ninja ID Card - Flutter App

A clean and minimal Flutter UI project that displays a stylish Ninja ID card with profile details.

---

## ✨ Features

* 🎨 Dark themed UI
* 👤 Profile avatar using assets
* 📛 Name and Ninja level display
* 📧 Email section with icon
* 📐 Clean layout using `Column`, `Row`, `Padding`, and `SizedBox`
